Конвеєрні системи для ящиків

Швидке та безпечне транспортування

Конвеєрні системи ASKO для ящиків застосовуються у разі необхідності транспортування та розподілу вантажів вагою до 80 кг.

Завдяки модульній структурі та характеристикам конвеєрів, що можуть змінюватися в широкому діапазоні клієнт має можливість отримати конвеєрну систему, яка буде на 100% відповідати його потребам. Під модульною структурою конвеєрної системи мається на увазі те, що система складається з окремих блоків, які можуть між собою конфігуруватися для виконання поставлених задач по транспортуванню вантажів.

  • роликові конвеєри з приводом 24VDC з функцією акумуляції ящиків
  • роликові конвеєри з приводом 400 VAC
  • горизонтальні стрічкові конвеєри
  • нахилені стрічкові конвеєри
  • ліфти
  • спіральні конвеєри
  • модулі для формування стопок ящиків
  • модулі для злиття потоків вантажів
  • модулі для розділення потоків вантажів

Широкий діапазон характеристик модулів це можливість змінювати конфігурацію кожного конвеєра, виходячи із умов проекту. Наприклад висота, ширина, довжина, швидкість, прискорення, алгоритм роботи та набір аксесуарів.

Складові компоненти

До основних модулів конвеєрних систем ASKO відносяться наступні компоненти:

  • роликові конвеєри з приводом 24VDC з функцією акумуляції ящиків
  • роликові конвеєри з приводом 400 VAC
  • горизонтальні стрічкові конвеєри
  • нахилені стрічкові конвеєри
  • ліфти
  • спіральні конвеєри
  • модулі для формування стопок ящиків
  • модулі для злиття потоків вантажів
  • модулі для розділення потоків вантажів

Переваги системи:

  • Швидке та безпечне транспортування вантажу. Функція нульового тиску дозволяє транспортувати вантажі, забезпечуючи дотримання відстані між ними.
  • Гнучкість. За рахунок модульної структури конвеєрна система забезпечує гнучкість під час реалізації проекту та можливість проведення змін в процесі роботи обладнання.
  • Мінімальні затрати на володіння. Завдяки застосування децентралізованих приводів конвеєрні системи ASKO дозволяють мінімізувати затрати електроенергії та мінімізувати вартість робіт обслуговування за рахунок використання виключно якісних компонентів.
